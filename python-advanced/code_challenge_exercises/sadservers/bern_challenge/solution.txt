You can add an alias when you connect the mariadb container essentially assigning it an additional DNS name

The container mariadb is still reachable as:

    mariadb

But now it is also reachable as:

    mysql

docker network create wordpress_net
docker network connect wordpress_net wordpress

# add network alias
docker network connect --alias mysql wordpress_net mariadb
